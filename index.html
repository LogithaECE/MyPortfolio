Featured Projects

MODULE 01 · EDGE-AI

EEG-Based Drowsiness Detection Helmet for Aviation Crew

Edge-AI brainwave monitoring & on-device cognitive state classification

Designed a smart aviation helmet that classifies cognitive state in real time from EEG signals, using a preprocessing pipeline and a Keras-optimized neural network. The trained model was converted to TensorFlow Lite and deployed directly on an ESP32 for on-device inference - triggering a buzzer and sending cloud alerts the moment fatigue is detected. The firmware was engineered for stable ADC sampling, low-power operation, and reliability in the field.

Deep Learning ESP32 Signal Processing TensorFlow Lite
MODULE 02 · IOT SAFETY

Biometric-Based Smart Motorcycle Safety System for Women

ESP32-CAM · GSM/GPS · fingerprint authentication · Telegram alerts

Built a safety system that authenticates the rider via fingerprint sensor before triggering any emergency response - combining GPS tracking and GSM-based SMS alerts to registered contacts. Live camera feeds stream to a Telegram bot with local SD-card backup, while an OTP-secured deactivation flow keeps the whole system usable without depending on a smartphone.

ESP32-CAM GSM/GPS UART I2C WiFi
MODULE 03 · BIOMEDICAL IOT

IoT-Based Pacemaker Monitoring System

Analog signal acquisition · MQTT · AES encryption · ThingSpeak dashboard

Captured analog ECG and SpO2 signals via biomedical sensors and digitized them in real time through a microcontroller. Patient data is transmitted over MQTT with AES encryption to keep medical telemetry secure, feeding a scalable pipeline that powers both a ThingSpeak dashboard and a custom HTML/JS visualization layer.

MQTT AES Encryption IoT Signal Acquisition
